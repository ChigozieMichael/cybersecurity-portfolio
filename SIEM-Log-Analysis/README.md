# SIEM Log Analysis (Wazuh or Splunk)

## Objective
Perform log collection, parsing, alerting, and detection using an open-source SIEM to simulate SOC Tier 1 monitoring tasks.

## Tools Used
- Wazuh or Splunk Free
- Windows Event Viewer
- VirtualBox

## Lab Setup
- Host OS: Windows
- VM: Kali Linux and Windows
- SIEM: Installed on Ubuntu VM or Wazuh all-in-one

## Procedure
- Installed and configured SIEM
- Connected Windows logs
- Created dashboards and detection rules
- Triggered brute-force login attempts
- Observed alerts in real-time

## Findings
- Detected multiple failed login attempts
- Identified privilege escalation events
- Analyzed Windows Security logs
