# Incident Response: Phishing Attack Simulation

## Objective
Simulate a phishing compromise and document full incident response aligned with NIST 800-61.

## Tools Used
- Python SMTP server or online phishing simulator
- Windows VM
- SIEM for detection

## Scenario
Simulated a phishing email that captured user credentials.

## Procedure
- Delivered phishing email
- User opened malicious link
- Captured logs showing unusual authentication behavior
- Performed containment, eradication, and recovery steps

## Deliverables
- Full incident report (PDF)
- Screenshots of logs and SIEM alerts

## Key Learning
- Understanding incident lifecycle
- Investigating phishing behaviors
